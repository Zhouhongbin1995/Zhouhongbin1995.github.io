Intro HTML and CSS

Posted on 2020-06-05 Edited on 2020-06-12 In CS

HTML

HTML称为超文本标记语言，是一种标识性的语言。它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。

非常感谢freeCodeCamp以及W3school的HTML总结!

HTML (Hyper Text Markup Language)不是编程语言，而是义中标记语言（markup language），用于描述网页结构。HTML标签通常是成对出现的，<>包围关键字称为开放标签，并用含/的标签表示结束关键词表示闭合标签。Adobe Dreamweaver或者Notepad都是非常不错的编辑器供使用，对于初学者，推荐一些轻量级的notepad更为优秀，进阶版可以选择VScode。

从html的一般模板来看，所有的元素都是被一件一件定义的，并且如文本，可以在两侧线性定义其属性。因此，在html中，如何线性增加网页内的元素是着实基础并且非常关键的内容，下一章节也主要总结了，有哪些关键字可以被使用，达到某种效果。HTML网页不同结构处的常用关键字。与CSS的联动可以让设计HTML 布局更为轻便。

CSS

层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。参数化系统地设置样式。

样式定义如何显示 HTML 元素，样式通常存储在样式表中，把样式添加到 HTML 4.0 中，是为了解决内容与表现分离的问题，外部样式表可以极大提高工作效率，外部样式表通常存储在 CSS 文件中，多个样式定义可层叠为一。

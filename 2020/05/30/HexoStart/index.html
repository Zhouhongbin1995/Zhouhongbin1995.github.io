HexoStart

This is for starting a new hexo blog based on github public service.

All the steps need to be executed line by line.

Out line:

  1. Setup the platform, nodejs, git and github account.
  2. Prepare local materials, theme or markdown file.
  3. A lot extra operation could be found on hexo official.

Setup First

node.js Setup

git Setup

node -v /This is for checking node version

npm -v /This is for checking npm version

git config --global user.name "your name"

git config --global user.email "xxx@xxx" /Typically, we need to set git’s user.name and user.email. Open git bash first

npm install -g cnpm --registry=http://registry.npm.taobao.org /Facile path to install cnpm

cnpm install -g hexo /Get hexo faster using cnpm

Prepare local materials

mkdir blog /Make blog file for all materials we need

hexo init /Initialize hexo

hexo s /Start hexo

hexo n first-paper /Creat a new md file

cnpm install --save hexo-deployer-git /Get deployer started

deploy:
type: git
repo: https://github.com/Zhouhongbin1995/Zhouhongbin1995.github.io
branch: master /Link public github with local files by git

hexo d /Deploy the files from local to public service

Congratulations，now you have your private blog!
